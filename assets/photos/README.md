# Photos

YOUR photos go here — none are shipped, on purpose. See MAKE-IT-YOURS.md §2.

- `covers/` — cover-grade shots (polished or striking)
- `lifestyle/` — candid camera-roll shots, catalogued in `lifestyle/PHOTO-INDEX.md`

Import a camera-roll dump (HEIC→JPG + auto-rotate):

```
python3 tools/import-photos.py ~/Downloads/IMG_*.HEIC
```
